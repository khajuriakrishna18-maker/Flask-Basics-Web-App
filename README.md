# Flask Basics Web App

A simple Flask application built while learning the Flask framework.

## Features

- Home route (`/`)
- Index page using HTML templates
- About page
- Form submission using GET and POST methods
- Dynamic user input handling

## Technologies Used

- Python
- Flask
- HTML5

## Project Structure
project/
│
├── first.py
│
└── templates/
├── index.html
├── about.html
└── form.html

## Routes

| Route | Description |
|---------|-------------|
| `/` | Welcome message |
| `/index` | Index page |
| `/about` | About page |
| `/form` | Form submission page |
