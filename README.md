# Flask Fundamentals 🚀

A structured collection of beginner-friendly Flask projects and examples created while learning the Flask framework.

This repository demonstrates the core concepts required to build web applications using Flask, including routing, template rendering, form handling, Jinja templating, and REST API development.

---

## 📚 Topics Covered

### 1. Routing and Templates

Location:

```text
01-routing/
```

Concepts:

* Flask routes
* URL endpoints
* HTML template rendering
* Form handling
* GET and POST requests

Files:

```text
01-routing/
├── app.py
└── templates/
    ├── index.html
    ├── about.html
    └── form.html
```

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/khajuriakrishna18-maker/flask-fundamentals.git
```

### Install Dependencies

```bash
pip install flask
```

### Run a Project

Navigate to any section:

```bash
cd 01-routing
python app.py
```

or

```bash
cd 02-jinja
python app.py
```

or

```bash
cd 03-rest-api
python app.py
```
